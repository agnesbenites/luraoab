# Lura OAB Landing — React / Next.js

Landing page em Next.js para o app Lura OAB, com estética clean, profissional e editorial minimalista.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Estrutura

- `src/app/page.tsx` — composição principal
- `src/components/sections/*` — seções da landing
- `src/components/ui/*` — botões, eyebrow, logo e SVG hero
- `src/app/globals.css` — design tokens, paleta, dark mode e base global
